### MessageMe - real time chat on websockets with friendly UI.

**Ruby & Rails version**
- ruby 2.6.3
- rails 5.2.4.1

**Technologies and features**
- Action Cable
- Form Validation
- Strong Password (bcrypt gem)
- 100% user friendly

**Frontend**
- Semantic UI

**Initial:**
```
git clone
bundle
rails db:create
rails db:migrate
rails db:seed
```
